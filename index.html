Heartland.US

For five years I acted as the primary web developer and technical expert for Heartland.US, a Global Payments company, utilizing my skills in HTML, CSS/SCSS, JavaScript/TypeScript, Scriban, C#/.NET, Bootstrap, jQuery, etc. to build pages and custom components in Sitecore CMS according to stakeholder requirements. I became and expert in building scalable, flexible, and complex components using the Sitecore Scriban markup language, which compiles to C# on the back-end.

The Entrepreneur's Studio

I developed a dynamic site for Heartland's podcast, The Entrepreneur's Studio, using primarily Scriban to create a site-within-a-site with its own Sitecore design theme, page templates, and components.

Dynamic Content Hub

A custom Sitecore solution using Sitecore CMS and Scriban with MVC coding best practices to dynamically assemble the HTML and populate each listing component on the page with tagged content from elsewhere on the site.

Sales Chatbots

I led the implementation and continuous development of Drift.JS chatbots on Heartland.US to acquire new sales leads and funnel them to the appropriate Salesforce teams. When Heartland fully merged with their parent company, Global Payments, I led the months-long effort to implement Drift on that site as well, working with many different stakeholders to finalize requirements, targets, etc.
